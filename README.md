## Licencia

Este repositorio utiliza la licencia **MIT**.

Elegí esta licencia porque permite que el código pueda ser utilizado, modificado y distribuido libremente, facilitando el aprendizaje y la reutilización del contenido.  
la licencia MIT es simple y flexible, lo que la hace ideal para proyectos académicos y personales, sin imponer restricciones complejas, manteniendo al mismo tiempo el reconocimiento al autor original.
